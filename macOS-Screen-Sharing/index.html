macOS 之间远程桌面 (screen sharing)

发表于 2016-11-23  |  2017-01-10

这里说的远程桌面是指：两台位于不同网络之间的 Mac 电脑进行互联，从而在 Mac A 上操作 Mac B，或反之。

  • 如果是「相同网络内访问另一台 Mac」，请看这篇帮助
  • 如果是「不同网络访问自己的另一台 Mac」，请看「回到我的 Mac」功能
  • 其他需求请看这篇概述

基本上，我们可以通过 macOS 内置的 VNC 功能实现各种屏幕分享需求。下面具体讲解如何远程桌面连接他人的 Mac（或让他们连接自己的 Mac）。

所需软件

  1. 两台 Mac 都搭载了最新版的 macOS（最少 10.10 或以上）
  2. 两个 Apple ID（如果只用一个，那便是「回到我的 Mac」功能）
  3. 两台 Mac 都分别用 Apple ID 登入了 iMessage
  4. 顺畅的宽带网络

理论上，我们不需要下载任何第三方软件。

步骤

  1. 打开 iMessage，选择想要远程桌面的好友，如果没有就新建对话
  2. 点击收件人右边的「详细」，在弹出窗口中选择分享屏幕
  3. 选择是要分享自己的屏幕，还是要连接对方的屏幕，然后等待对方接受请求



这个远程桌面实际是启动了 screen sharing.app 应用，因此我们也可以不通过 iMessage，而是直接打开该应用，输入对方的 Apple ID 来连接。如果用 Spotlight 则更加方便，还可以直接使用之间连接过的 Apple ID（系统会自动保存记录）。

其他注意事项

  1. 连接后只能看、不能操作，如需操作的话需在窗口左上角点击控制，然后等待对方允许
  2. 控制时，对方仍可进行操作，所以如果遇到不太方便的操作，可以让对方代劳，提高效率
  3. 除了屏幕画面以外，双方的声音也会通过麦克风接收并传输（类似视频通话），请注意
  4. 若还有其他问题，请查看官方帮助文章
