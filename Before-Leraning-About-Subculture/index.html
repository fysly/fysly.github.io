且慢接触亚文化

发表于 2015-12-23  |  2017-01-10

就好比读大学前要读高中、高中之前要读初中、初中之前要读小学一样，人应该在对某类主文化有足够了解之后再去深入亚文化。

譬如说吧，很多小朋友一上来就看各种深夜动画，但对子供向、全民向作品却所知甚少，主流三次元电视剧更是一窍不通；听音乐，一上来就听各种宅向アニメソング[1]，对声优演唱的曲目如数家珍，大牌歌手、乐队为动画献唱的反而所知甚少，对日本流行音乐更是一窍不通。而这样做所带来的后果，就是各种认知错误，导致各种令人无法直视的弹幕层出不穷。

上月那个动画歌曲公信榜单曲销量 Top100 可谓是此类弹幕重灾区，相当数量的人搞不清楚如下几个非常基本的事实：

  1. 不知道销量榜是什么（销量与歌曲传唱度不一定相符，有些歌只是在小众圈子里火，或是粉丝复数购买）
  2. 不知道公信榜是什么（公信榜只统计实体出版，很多人气歌曲人们会选择更加便宜方便的数字版，因此实体唱片销量高的基本是偶像团体，因为 CD 里有写真、握手券等附属品）
  3. 不知道单曲和专辑的区别（很多耳熟能详的经典歌曲并未发行单曲，而是直接被收录在某张专辑发售，自然没法上单曲榜）
  4. 不知道高排名歌曲的销量为什么高（因为是偶像团体的单曲，购买者绝大多数是粉丝，他们是为了自己的偶像而买，甚至可能根本没看过这部动画）

实际上，这个视频本身也很值得吐槽。入榜的很多歌分明是 B 面曲，之所以卖那么高并非因为这首歌本身，而是 A 面那首主打歌。当然，我们也不能指望如今这些没有买过 CD 的孩子知道 AB 面曲是什么概念。

这个视频只是冰山一角，实际上此类话题早已屡见不鲜，网上也有不少对宅群体的批判，从而产生诸如「冻鳗」[2]这样的调侃。

什么，你觉得我在秀优越感？请重新阅读本文第一段。为了解释这一现象，我特意拿上学作比喻——应该没人觉得读小学、初中比读大学更有优越感吧？


❧

  1. Anime Song，即动画音乐，包括片头曲、片尾曲、插入曲等正式出版发行作品 ↩

  2. 动漫一词本是动画漫画的合称，但有些小朋友却以为日本动画叫动漫，中国「喜洋洋」那种才叫动画，由此产生不少病句及滥用。比如看了一集动画却说看了一集动漫；别人问喜欢看什么动画，却回答说自己只看动漫，不看动画。「冻鳗」一词便是网友对此类小朋友的嘲讽。 ↩
