Chrome for Mac 使用技巧汇集

发表于 2016-12-31  |  2018-03-23

虽说是 for Mac，但不少内容在其他平台也适用。
本文内容随时更新。

  1. Retina 屏幕的 Mac 不需要 LCD 渲染[1]，除了系统偏好设置中可以关闭 LCD 渲染之外，Chrome 浏览器内也可以把 chrome://flags/#lcd-text-aa 关闭

  2. Chrome 非常自作主张地根据系统语言更改了默认字体，我们可以下载这个扩展来还原。首先把默认字体改为西文字体（这一点很重要[2]），再把繁简中文改成合适的中文字体

  3. Chrome 的地址栏会记忆用户输入过的地址，并在以后自动补全，但如果不小心记忆了错误的网址就反而弄巧成拙了，对时我们可以选中错误的地址，然后按 ⇧ + Fn + Delete 删除这条记录

  4. 勤用快捷键 ，例如 ⌘ + L 可以移动光标到地址栏并全选，然后便可以输入地址，全程无需鼠标操作

  5. 在上一条的基础上，使用扩展 Vimium 以实现更加方便的键盘操作，进一步缩小鼠标依赖

  6. 现在的 Chrome 越来越不稳定了，经常因为 Flash 等问题占用大量资源而卡死，对此可以打开任务管理器查看究竟是哪个扩展、标签页出了问题

  7. 自定义搜索引擎[3]是个非常重要且好用的功能。例如设置淘宝为 tb，当你在 Chrome 地址栏输入 tb 时，浏览器就会提示你按下 Tab 键直接进入搜索状态，然后再输入想要搜索的关键词，如 iPhone 7，即可快速进入淘宝搜索。

    实际上，这一功能并不只适用于搜索，还可以用来直接访问 RESTful 的网页。例如我们可以发现，维基百科词条的地址都是 https://zh.wikipedia.org/wiki/ 开头[4]，后面跟着关键词，所以同样可以利用自定义搜索引擎，快速打开某个词条。比如将危机百科设置为 wiki，按下 Tab 后再输入 Google 并回车，即可快速进入谷歌的危维基百科词条。

  8. 对于一些不希望留下使用痕迹，尤其是不希望被记录隐私的网站，可以使用隐身模式。比如淘宝，如果不用隐身模式搜索关键词，一定会被精准地记录，并推荐相关商品。

    隐身模式可以在右上角选项中打开，也可以直接用快捷键⇧+⌘+N打开。

  9. Chrome 的搜索框有许多功能，可以快速显示某些结果而无需按下回车键，除了许多人都知道的计算器以外，甚至还能起到辞典的作用。
