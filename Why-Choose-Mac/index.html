为什么选择 Mac

发表于 2016-07-06  |  2017-01-10

准确说，我选择的是一台 MacBook Pro (Retina, 15-inch, Mid 2015)，总共花费 11550 人民币（港版）。虽然购买这台电脑已过于一年多时间，但现在仍然值得回顾一下，来具体说说我为什么要花这 11550 人民币买这台机器。

硬件配置

屏幕

这是一块 2880x1800 的 Retina 屏幕，这个分辨率在这个尺寸上非常合适。由于 macOS 系统以 1400x900 的大小显示，因此并不会出现「由于窗口、图像的分辨率低，在最高分辨率下变得非常小」的糟糕体验。但并不意味着我们得到的是 1400x900 的显示效果，因为这是一块真正的 2880x1800 分辨率显示屏，实际上苹果把 4 个像素点合成了一个，以这种方法使图片、文字的锯齿消失，看上去清晰了四倍，同时又保持了原来的大小。

同时，这款屏幕的色域为 100% 的 sRPG，不算特别高，但足够用。折算 NTSC 相当于 72% NTSC 色域，作为对比，某高性价比品牌的最新系列游戏本，屏幕色域仅有 45% NTSC。实际上这并非个例，绝大部分非高端笔记本的屏幕色域都很低，以至于画面总是泛白。因此，想要好颜色，就必须花大价钱买高端笔记本。

硬盘、内存、处理器

我们都知道，SSD 比 HDD 快好几倍，而 PCIe 接口的 SSD 又比普通 SATA 接口的快几倍。这款 MacBook Pro 采用的，正是前者。在这一点上，其实并没有多少竞争对手，若不舍得花这钱，你很难获得同样的高速体验。

同时，MacBook Pro 15-inch 最低配置就是 16G 内存，不可再降低；处理器则采用标准电压的 i7 处理器，虽然频率不是很高。综合来看：PCIe 的 SSD，16G 内存，i7 处理器，同时具备这三样的笔记本真的不多，其实并没有多少竞争对手。很多人都说苹果的产品贵，但我们也知道：越高档的产品越贵（性价比越低），因为越高档，成本就越高，最低价格就越高，销量就相对比较少。为了获取足够的利润，商家会进一步提高价格。相反，如果是低端产品，由于成本低，销量高，商家可以薄利多销。

换言之，高端的产品由于大家都贵，苹果也就不显得贵了。想要凑齐上述全部配置，即使是其他品牌也不会便宜。那为什么不选择外形更加好看（Unibody 的镁铝合金机身）、支持 macOS 的 MacBooK 呢？

毫无疑问，除了显卡，这台电脑各方面都相当出色，可以满足多种需求。而操作系统方面，macOS 又可以与硬件配置相得益彰。

软件体验

网页浏览

我相信绝大多数人闲暇时间花在电脑上最多的时间，便是浏览网页，包括上微博、看在线视频等。浏览器方面，四大浏览器中，Chrome 和 Firefox 是全平台支持，macOS 毫无问题。苹果自家的 Safari 也颇具特色，简洁流畅。只要你不是 IE 拥趸，切换到 macOS 完全无影响。

大内存、高速硬盘、多核处理器，带来的是极佳的浏览器体验，再也不用担心 Chrome 卡顿了。同时 macOS 出色的字体渲染效果也和 Retina 屏幕相得益彰，文字锐利而准确，极大程度的满足了眼睛。

这一点值得说明。一般认为，Windows 的文字渲染主张「为了清晰显示文字，不惜对字体字型进行破坏」，而苹果则主张：「字体该是什么样，就显示成什么样，越接近印刷显示效果越好」。我更赞同苹果的思路，因为字体之所以模糊，本部原因是屏幕分辨率不够，那么解决方案毫无疑问，应该是提高屏幕分辨率，因此，苹果先是推出 iPhone 4，再是 Retina 屏幕的 MacBook Pro，最后是 iPad 和 iMac，逐步使全线产品都用上 Retina 屏幕，从根源上解决了字体显示问题。

而微软则过于极客了，通过软件渲染的方式，扭曲了字体字型，强行使其变清晰，又不对高分辨率提供更好的支持，直到 Windows 10 仍然没能完美，不得不说令人遗憾。绝大部分的用户使用电脑，接触最多的就是文字，文字显示效果将极大影响体验。因此，我建议非工作用电脑（工作用电脑则根据公司、工作要求决定），如果不玩游戏的话，都可以考虑购买 Retina 屏幕的 Mac。

举个最近遇到的例子吧：某 Git 图形化工具 Tower 的官网下方有展示应用的截图，其中 Mac 页面是 Mac 版应用的截图，而 Windows 页面则是 Windows 版。对比这两个版本的截图，我们可以很清楚地感受到它们的字体效果差距。

文字编辑、写代码

作为一款 UNIX-like 操作系统，macOS 对开源世界的支持程度很高，可以使用很多开源项目、库，也有很多优秀的编辑器、IDE，可以说是非 Windows 程序员的最佳编程用电脑。而对于普通文字编辑者、作家，由于 macOS 拥有大量 Windows 都没有的写作软件，再加上出色的文字显示效果，它自然也是这一人群的最佳选择。

总结

我平时使用电脑，花时间最多的就是在浏览器里访问各种网站，看大量的文字（需要高清屏幕和好的文字渲染效果），浏览器用的是 Chrome（需要大内存和高速的 SSD）；其次是上在线视频网站观看大量视频（需要高清且色彩丰富的屏幕）；其次是写文章（需要优秀的写作软件和好的文字显示效果）；再次是折腾很多开源软件，满足各种需求（需要 macOS，相当于拥有好用的桌面的 Linux 系统）。

我对电脑的需求就是满足上述需求，而且要快，越快越好。因此，这台 MacBook Pro 是最适合我的选择，它没有独立显卡（节省了钱、空间、重量、发热），但却有大容量内存、高速 SSD 和 i7 处理器，再加上这块出色的屏幕和 macOS 操作系统——完美。

如果你对 Mac 感兴趣或想买一台 Mac，看完这篇文章后应该明白 Mac 是否适合自己了（游戏党请止步）；如果你想买电脑，碰巧有和我相似的需求，但不知道买什么电脑好——那你现在应该知道了。
