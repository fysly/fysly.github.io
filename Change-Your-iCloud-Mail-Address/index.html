治标不如治本——更换 iCloud 邮箱

发表于 2016-09-23  |  2017-01-10

起因

澳门赌场广告有多烦人，想必大家都有目共睹。他们已经发展到能够利用邮件提供商来发送垃圾邮件了。比如利用 Gmail 的辅助邮箱功能，先将你的邮箱设为辅助邮箱，然后再修改，于是 Gmail 就会自动给你的邮箱发送确认邮件。由于他们把名字改成了很长的广告词，于是这封邮件实际上就起到了广告邮件的效果。另外还有 QQ 邮箱，同样使用了类似方法，让腾讯发送官方邮件给你，如果你本身就是 QQ 邮件使用者，连拒收、标记垃圾邮件都无法做到，因为发件人是腾讯官方。

同样饱受其害的还有苹果用户。由于 iMessage 对垃圾短信无法智能过滤（内容是加密的），只能由用户收到信息后再手动举报。但这并不是个办法，毕竟骚扰已经铸成，所以不少人只能彻底关闭 iMessage 功能，杜绝后患。

但并不是退一步就一定能海阔天空的。通过上述的邮件例子，我们可以明显感受到对方的「强大」，果然，他们又想出了新花样：通过日历邀请来推送广告。普通日历的邀请一般通过邮箱来发送，而苹果为 Apple ID 用户提供了直接推送到手机或电脑的功能。这一原本提升用户体验的功能，如今却成为了对方的利用工具，虽然我们还有办法解决。

但并不是解决一次就能完事，最近微博又出现了澳门赌场广告的新招——通过 iCloud 照片共享来推送广告。哎，国人总是这些方面充满了智慧。

那，到底该怎么办？

更换 Apple ID 邮箱

通过上述例子我们不难发现，之前对付这种广告的办法效果都很有限。原本，这些功能是苹果为了方便用户而推出（为了节省短信费，推出了走流量的 iMessage；为了接受邀请更快捷，将日历邀请直接推送到设备；为了方便分享照片，推出 iCloud 照片共享……），结果却成了垃圾广告的温床。关掉其中一个功能，只会让对方将战场转移到下一张温床，直到你把所有功能全都关闭。所以，我们必须使用更加有效的办法。

其中一个比较可靠的是更换 Apple ID 主邮箱。澳门赌场广告，其实和过去的垃圾邮件并没有什么本质区别。如果我们更换一个更好的邮箱，使对方发送邮件的成本提高，无法直接群发，便有机会躲过这些骚扰了。

这里建议使用 Outlook 邮箱，它可以自由添加多个别名（也就是邮箱地址）。我们可以添加一个专门用于 Apple ID 的别名，除了苹果和你自己，谁也不知道。

Gmail 也有别名功能，但做得比较特殊：别名是以你的原 ID 后面添加 +xxxx（xxxx为任意可用字符）来形成，理论上有无数个。例如 ID 为 jameswang 的谷歌账号，原 Gmail 地址是 jameswang@gmail.com，那么便可以使用诸如 jameswang+abc@gmail.com 或是 jameswang+appleid@gmail.com 等别名。这些别名无需额外设置，直接就能使用（你可以直接尝试发送一封邮件给它，然后在主邮箱中接收）。

Outlook 的别名优点在于可自由设定，缺点则是数量有限（其实也足够了）；Gmail 的别名由于其特殊性，数量无限，但也正因为这一特殊性，别名比较长、丑，且他们一眼就能认出这是别名。

总的来看，我还是推荐用 Outlook，而且它对垃圾邮件的审核更加严格，很多正常网站的邮件也会被 Outlook 当作垃圾邮件处理，且无法设置修改。这原本是一个很麻烦的缺点，但在这种情况下反而成了优点。

你可以登入 Apple ID 官网，点击账号右边的编辑按钮，然后选择编辑电子邮件地址。

重新注册一个 iCloud 邮箱

除了给 Apple ID 更换邮箱这一方法外，其实还有一个更加偏门的办法，那就是重新注册一个新的 Apple ID。注意，重新注册的 Apple ID 是使用在 iCloud 上，而不是 App Store，所以无需担心原本已购买的内容。在使用新 ID 登入 iCloud 后，我们仍然可以继续将旧 ID 登入 App Store 以下载已购买的应用和其他内容。这是完全可行的。

之所以重新注册，是为了获得一个真正的 Apple ID。细心的用户可能已经发现，其实 iCloud 也自带了邮箱功能，但若想要使用，则必须「激活」一个新的 @icloud.com 邮箱。除了原邮箱地址外，我们也可以用这个 @icloud.com 邮箱登入 Apple ID。但实际上，这两个地址背后并不是同一个 ID。当我们为了打开邮箱功能而激活 @icloud.com 邮箱时，其实相当于又注册了一个新的账号，系统会在内部把两个 ID 绑定，当作同一个 Apple ID 来识别对待。

处女座或强迫症患者看到上述文字以后应该已经明白为什么要重新注册 Apple ID 了。不为别的，正是为了不使用已有的第三方邮箱，而是直接注册 iCloud 邮箱，避免上述两个 ID 绑定成一个的「不优雅」解决方案。

iCloud 邮箱的好处有很多。首先它是大厂苹果出品，安全性毋庸置疑，且在苹果设备上使用十分方便（登入 iCloud 后就可以直接打开原生的邮箱应用使用）。相比 Gmail 又没有众所周知的网络连接问题。我们可以完全将其作为一个私人用邮箱，只在 iCloud 和 iMessage 等服务中使用，以及给联系人里的熟人发邮件分享各种信息。

另外，由于 iCloud 邮箱使用者不多，只要自己不随便泄露，几乎不可能受到垃圾邮件的骚扰。同时，也正因为使用者不多，我们目前仍然可以申请较短或较个性的 ID。

目前只能在苹果设备（macOS 或 iOS）上注册 iCloud 邮箱。二者都是在 iCloud 设置处，登出（注销）原账号后，选择注册，填写出生日期、进入下一步后，再选择「使用免费的 iCloud 邮箱」，便可以进入注册页面了。

注册完成后，便用这个新邮箱登入 iCloud，和骚扰信息说再见。

补充

更换了 iCloud 账号，难道要把原来账号里的联系人、日历等数据都丢掉？当然不是，这些数据其实都还好好地躺在原账号或本机里。

在登出账号时，我们可以将这些数据保留在本机，然后再登入新账号；也可以直接导出备份（macOS 上导出比较方便），登出时从本机删除，重新登入新账号后再导入。

我们可以用各种不同的办法来迁移数据，无需担心其丢失。
