macOS 中文输入法之选：Rime

发表于 2016-04-23  |  2017-01-15

关于 macOS 软件之选

我打算用这个系列，介绍一些非常好用、但一开始可能有点难上手的软件。其中绝大部分我都在「macOS 必要设置指南」里提到了，但由于篇幅、时间以及个人原因没有展开，所以准备重新在这个系列里写一下，也算解答身边新买 Mac 的朋友之疑问了——既然我安利你们买 Mac，自然不会在这种问题上让你们失望。

这篇讲中文输入法。

为什么用 Rime

虽然 Mac 缺乏很多应用软件，但中文输入法相对来说情况好很多，毕竟对中国人来说这是最基础的软件，各大厂商也没有松懈，不敢轻易放弃 1% 的 Mac 用户。因此，中文输入法的选择其实并不算少，诸如搜狗、百度等都有 Mac 版推出，更新也比较勤快。

但同时我们也知道，国产软件往往会有一些「毛病」，虽然在 macOS 上程度一般会有所收敛，但如果有更好的选择，我们自然应该毫不犹豫地放弃这些大厂。而中文输入法这一领域，恰好就有这么一款足够好的开源作品，可以让你抛弃搜狗百度，它就是 Rime 输入法。

Rime 输入法的中文名字十分有趣：在 Windows 上它叫小狼毫，在 macOS 上它叫鼠须管，在 Linux 上它叫中州韵，充满文艺感。但我们姑且先不管作者为什么要把同一个软件在不同平台起不同名字，至少这里我们得到了一个十分重要的信息：Rime 输入法居然同时支持三大桌面操作系统！

这当然是它的优点之一：在 macOS 上我们可以用它，在 Windows 上我们也可以用它，使用体验是可以无缝切换的。但，多平台也不是其最主要、最值得夸耀的特征，毕竟这个系列我们只关心 macOS 平台。Rime 真正名声在外、令人称奇的地方在于：它居然支持十几种输入方案！

不止是我们最常用的拼音，还有注音、仓颉、速成、五笔、双拼、粤拼、吴语、中古汉语拼音、五笔画、Emoji、国际音标、宫保拼音……没错，这么多输入方案，居然真的被集成在一款输入法之内了。

这还不是全部，最关键的一点是：Rime 是开源软件，源代码对所有人开放，绝不会做什么小动作，更没有云服务功能，不会把任何数据从你的电脑上带走。输入法是非常隐私、重要的工具，我们每天会用其输入大量不可外泄的内容，自然是越安全越好。

结合以上三点，还不足以让我们选择它吗？

Rime 其他优缺点

Rime 当然还有其他优点，毕竟输入法最重要的还是输入效率。首先，由于它没有乱七八糟的功能，比起主流输入法小巧多了，同时也增加了稳定性，因此其使用体验十分流畅，打字如行云流水。这一点，打字越快的人越能体会。一旦输入法不够快，很可能出现屏幕显示跟不是手指敲击的频率，影响输入准确率。

其次，它的配置选项足够丰富，我们可以对其进行充分地定制（包括外观），打造专属于自己的输入法，足够独特，也足够顺手。

当然，它也有缺点。比如，配置文件虽然提供了丰富的个性化定制，但也提高了使用门槛，普通人可能真不知道该怎么设置它。因此，我会在这篇文章里作简单介绍。再比如，由于作者是香港人，Rime 软件本身、以及除五笔外的所有输入方案都是繁体中文（香港），只有拼音输入提供了另外两种方案：简体和台湾正体。如果你对繁体中文有厌恶心理，那么 Rime 可能就不适合你。

不过我们无需担心简体中文的输入问题，一方面拼音有简体中文方案（相信绝大部分人都是直接使用拼音吧），另一方面，由于 Rime 内置了极为强大的繁简切换功能，可以在设置中切换成简体输入模式，即使是其非拼音方案，也可以输入简体字。之所以说强大，是因为这个切换功能是根据词汇为基础的，从而避免了「一简对多繁」时可能出现的错字问题（还记得毕老爷「玖球天後」的笑话么）。

Rime 的安装、设置

首先从官网下载，然后解压缩、安装。然后切换输入法至 Rime，点击菜单栏的输入法图标，选择「用户设定」，便会打开 Rime 的文件夹，里面是一些配置文件和词库。为了修改这些配置文件，请先确保已有一款合适的文本编辑器（如 Sublime Text）。

关于设置，主要是 Rime 本身和不同输入方案的设置。其中，带 Dafault 字样的文件是系统默认设置，我们不要直接修改它，而是在另外的 custom 文件中（如果没有，则根据已有的，仿照其命名方式新建）添加我们的设置。这样的好处是日后输入法升级，并不会修改我们的 custom 文件，因此我们的自定义设置也就不会丢失了。

Rime 本身的设置

这里，主要是外观，也就是输入法的选词框长什么样，是横着的还是竖着的，圆角半径多大，什么配色；候选词用什么字体、字号多大，一共显示几位。

其次是使用的输入方案。虽然 Rime 支持那么多输入方案，但我们肯定只使用其中几项，那么就不必打开全部方案，避免方案过多，切换起来很浪费时间。例如，我就只打开了双拼方案（是的，我就只用双拼！）。

不同输入方案的设置

这里可以设置你使用的输入方案的具体内容，比如你主要就用拼音，那就打开拼音的 custom 文件，进行个性化设置。例如可以设置拼音输入时，不同的标点符号按键默认的输出字符（比如顿号如何输入、省略号如何输入、引号设置成直接输出直角引号「」）。

关于配置，由于我用的是双拼，且比较个性化，就不贴出自己的配置了。不知道该如何配置的同学建议看这篇文章学习借鉴一下，作者还提到了增加词库的方法。但必须建议不要全盘照搬他人的配置，因为这些配置往往十分复杂，且涉及诸多内容，对于普通人来说可能其中不少都是不需要的，请仔细看配置里每一项后面的注释文字，看它是做什么用的。同理，词库也是一样，并不是所有词库都对你有用。

Rime 的使用

你可以在系统环境设置中增加、删除 Rime（但不会删除软件本身），如果安装完 Rime 发现无法使用，可以打开设置看看是否已经启用。其次，每次修改了配置后，都必须点击菜单栏的 Rime 图标，选择「重新部署」，才能使配置修改生效。
