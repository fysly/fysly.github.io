macOS 视频播放器之选：mpv

发表于 2016-05-23  |  2017-01-10

关于 macOS 软件之选

我打算用这个系列，介绍一些非常好用、但一开始可能有点难上手的软件。其中绝大部分我都在「macOS 必要设置指南」里提到了，但由于篇幅、时间以及个人原因没有展开，所以准备重新在这个系列里写一下，也算解答身边新买 Mac 的朋友之疑问了——既然我安利你们买 Mac，自然不会在这种问题上让你们失望。

这篇讲视频播放器。

为什么选择 mpv

OS X 上不是没有简单易用的视频播放器，但它们总有这样那样的缺点，几乎没有哪一款能达到「只用它一个就够了」的程度。Movist 可能是最接近这个要求的，但仍然有明显缺点，再考虑到它 5 美元的售价，还不如直接用免费开源的 mpv 得了。

mpv 脱胎于大名鼎鼎的 MPlayer 和 mplayer 2（目前各大操作平台最好的播放器几乎都和其有关），并可以说做到了青出于蓝。虽然它没有 GUI 的设置界面，但提供了非常细致的配置文件和文档，可以进行相当细微的设置，满足各种需求。通过配置文件设置虽然不够直观，但好在并不难懂，稍微了解一番即可，只要保存了配置文件，便可以讲设置保存，还可以多平台、不同设备使用。

配置

如果你的英语比较好的话，可以直接去看 mpv 的官方文档，上面写得非常详细，你可以根据自己的需求配置，但实际上绝大多数内容无需设置，默认就可以，只有少数几个是必须设置的，由于是涉及不同操作系统、硬件配置的。例如，macOS 用户必须启用 fs-black-out-screens 一项，否则 mpv 播放时会卡顿。再比如，Retina 屏幕的 Mac 电脑应该设置 window-scale=0.5，将视频默认以一半大小显示，这样才是最清晰的，而且不会因为分辨率过大而满出屏幕。之后可以考虑全屏播放其其他大小。

使用

如果你使用了 --with-bundle 参数，并使用 brew linkapps mpv 命令，那么就可以像普通应用程序一样使用 mpv，首先可以设置不同格式文件的默认打开方式，将 .mp4 或 .mkv 等文件都默认以 mpv 打开。

打开视频后，mpv 就会启动，你会看到一个窗口，里面是视频内容，除此之外就什么也没有了，非常简洁（如果你使用了我的配置，里面把窗口边框也去掉了，更加简洁）。

然后，所有的操作都通过快捷键来完成，例如：快捷键 F 是切换全屏播放 (Full screen)，方向键是快进、快退。如果有疑问，可以参考官方文档。

如果在使用过程中遇到问题，多半是配置和目前操作系统和硬件不搭配，请多利用谷歌搜索，寻找相同案例，从而获得解决方案。

总结

mpv 不仅是 macOS 平台上最好的播放器，即使在 Windows 上也并没有什么对手（除了缺乏 GUI 界面这一点会让很多人觉得难以使用以外），推荐所有人尝试。而对于 macOS 用户，它则具有更多意义，因为 macOS 上的好软件大多都是付费的。既然有免费且好用的选择，那就可以不去使用付费软件。

我之所以强调这一点，更多是希望「对自己的需求不了解，到处乱下盗版，开发商受到损失、盗版商白赚流量和广告」的情况能够减少。

这不是一个好的现象。
